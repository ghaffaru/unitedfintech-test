# unitedfintech-test
> Employee management api

## Installation
Use docker to run

```
docker-compose build
```

Wait for docker to do its thing

## Usage
Create the admin user
```
docker-compose run web python manage.py addAdmin
```

### Run the application
```
docker-compose up
```
